dcmri 0.6.16
============

Summary
-------

New in this version:

- New tutorial on inflow effects, using minipig data
- Update kidney interfaces with improved documentation
- Define new .dmr format for ROI-level data along with read and write functions

Changes
-------

- Introduce dmr format + update kidney & aorta interface + new tutorial (`#205 <https://github.com/dcmri/dcmri/pull/205>`_).
- update version number (`#206 <https://github.com/dcmri/dcmri/pull/206>`_).
- include minipig data (`#207 <https://github.com/dcmri/dcmri/pull/207>`_).
- weaken numerical test to avoid rounding errors (`#208 <https://github.com/dcmri/dcmri/pull/208>`_).

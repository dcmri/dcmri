dcmri 0.6.11
============

We're happy to announce the release of dcmri 0.6.11!


Highlights
----------

- 3 new use cases (liver and kidney)
- User guide extended with details for multi-compartment signal models
- New models added including inflow and partial volume correction in liver models

Enhancements
------------

- feat: add use case for TRISTAN reproducibility study (`#168 <https://github.com/dcmri/dcmri/pull/168>`_).
- feat: add use case for TRISTAN multiple dosing study (`#169 <https://github.com/dcmri/dcmri/pull/169>`_).

Documentation
-------------

- Improvements in README, User guide structure and whole-body guide (`#163 <https://github.com/dcmri/dcmri/pull/163>`_).
- Documentation improvements in README, user guide headings and whole-body models (`#164 <https://github.com/dcmri/dcmri/pull/164>`_).
- Documentation extended with steady-state signal models (`#165 <https://github.com/dcmri/dcmri/pull/165>`_).
- Documentation extended for restricted water exchange (`#166 <https://github.com/dcmri/dcmri/pull/166>`_).
- Update README.rst (`#176 <https://github.com/dcmri/dcmri/pull/176>`_).

Other
-----

- Bug correct in liver export + small changes in user guide docs (`#167 <https://github.com/dcmri/dcmri/pull/167>`_).
- Harmonize kidney models and draft kidney example (`#170 <https://github.com/dcmri/dcmri/pull/170>`_).
- New models and examples: SPGR, magnetization, and liver dual inlet with flow correction (`#174 <https://github.com/dcmri/dcmri/pull/174>`_).
- harmonized dual-inlet liver models (`#175 <https://github.com/dcmri/dcmri/pull/175>`_).

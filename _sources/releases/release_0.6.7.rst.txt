dcmri 0.6.7
===========
We're happy to announce the release of dcmri 0.6.7!#

This release has an additional tutorial on AIFs, and a new use case (TRISTAN 6 compound). 

Additionally a new population AIF with tunable parameters, adn 2 new datasets available via fetch()

Enhancements
------------

- New datasets, use case, AIF and tutorial (`#125 <https://github.com/QIB-Sheffield/dcmri/pull/125>`_).

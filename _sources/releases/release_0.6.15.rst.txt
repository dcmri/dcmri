dcmri 0.6.15
============

Summary
-------

New in this version:

- Removed all *in construction* notices as the package is now more stable.
- Moved the data onto zenodo and removed from package to make the distribution 
  lighter.
- Added tracking of website activity to understand uptake and usage.
- Some renaming of renal functions.

- kidney cort med rename (`#200 <https://github.com/dcmri/dcmri/pull/200>`_).

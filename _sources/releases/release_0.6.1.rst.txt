dcmri 0.6.1
===========
We're happy to announce the release of dcmri 0.6.1!

Summary
--------

This is the first functional release with one section on :ref:`pharmoacokinetic building blocks <PK blocks>` complete, including full documentation for all functions and 100% test coverage.
